Learninghack

A bootsrap e-learning template with TinCan and SCORM support.

This template is usefull when you wish to make responsive content and want it to communicate with e-learning systems.

What can you do with this template?

The project consist of slides defined by the a div tag with 'slide' class name.

Inside a slidetag you can use bootstrap code to define the content

The content will then be responsive!

Why does it matter?

Not evereything is posible to make with an e-learning autheringtool. Therefore I have made this template
